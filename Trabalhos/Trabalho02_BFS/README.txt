A distância d ij entre qualquer par de vértices Vi e Vj de um grafo não dirigido pode ser determinada por meio de uma busca em largura. Seja M uma matriz tal que M ij = dij. Faça um código para realizar uma busca em largura, a saída do seu programa é a matriz M exibida na tela.

Para testar seu código foi disponibilizado um exemplo de teste com a saída esperada. No arquivo .in contém o nome do arquivo que deverá ser lido. O arquivo .out contém a saída esperada para aquele caso de teste. Se código deve ser executada da seguinte forma: 

código < case.in > resposta.out

A saída resposta.out ,deve ser igual ao arquivo case.out.
